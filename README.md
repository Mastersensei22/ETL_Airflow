# ETL_Airflow
Arquitectura ETL distribuida que consume datos desde una API externa, genera datos sintéticos para simular crecimiento, transforma la información y persiste resultados usando Apache Airflow sobre Docker con CeleryExecutor con almacenamiento local, generando
al final un archivo con la informacion necesaria lista para ser almacenada en un Datawarehouse o para ser usada en un proceso de BI.
